# CondoGest - Sistema Inteligente de Gestão de Condomínios

**CondoGest** é uma plataforma completa para administração de condomínios, desenvolvida para modernizar a interação entre moradores, portaria e administração. O sistema foca em segurança, transparência financeira e facilidade de uso.

---

## 🚀 Principais Funcionalidades

### 1. Painel Administrativo Moderno
- **Dashboard Intuitivo:** Visão geral de chamados, finanças e visitantes em tempo real.
- **Gráficos Interativos:** Acompanhamento visual do fluxo de pessoas e despesas.
- **Modo Escuro (Dark Mode):** Interface adaptável para maior conforto visual.

### 2. Controle de Acesso e Segurança (Portaria)
- **Leitor de QR Code:** Entrada rápida e segura para moradores e visitantes pré-autorizados.
- **Registro de Visitantes:** Histórico completo de quem entra e sai, com horários e destino.
- **Gestão de Encomendas:** Notificação automática para moradores quando uma encomenda chega.

### 3. Gestão Financeira e Transparência
- **Portal da Transparência:** Moradores visualizam gráficos de despesas do condomínio (água, luz, manutenção).
- **Emissão de Boletos/Pix:** Facilidade para pagamento de taxas condominiais.
- **Relatórios:** Exportação de dados financeiros para auditoria.

### 4. Convivência e Reservas
- **Reservas Online:** Agendamento de áreas comuns (Churrasqueira, Salão de Festas) com bloqueio automático de conflitos.
- **Classificados:** Marketplace interno para moradores comprarem e venderem itens.
- **Prestadores de Serviço:** Cadastro de funcionários domésticos (diaristas, babás) para liberação na portaria.

### 5. Manutenção e Zeladoria
- **Chamados:** Abertura de solicitações de reparo com envio de fotos.
- **Plano de Manutenção:** Calendário de tarefas preventivas para o Zelador.

---

## 🛠️ Tecnologias Utilizadas

- **Backend:** Python (Flask), SQLAlchemy (ORM).
- **Banco de Dados:** PostgreSQL.
- **Frontend:** Bootstrap 5, Jinja2, JavaScript (Chart.js para gráficos).
- **Segurança:** Hash de senhas (Argon2), Proteção CSRF, Proteção contra Força Bruta (Rate Limiting).
- **Hospedagem:** Render.com (Aplicação Web e Banco de Dados).
